# Argent Bank - Full Stack Application

## Structure du projet

- `backend/` - API REST avec Express.js et MongoDB
- `frontend/` - Interface utilisateur avec React

## Documentation API

La documentation complète de l'API est disponible au format OpenAPI/Swagger :

- **Fichier** : `backend/server/swagger.yml`
- **URL locale** : http://localhost:3001/api-docs (une fois le backend lancé)

## Installation et lancement

### Backend

```bash
cd backend
npm install
npm run dev:server
```

### Frontend

```bash
cd frontend
npm install
npm start
```
